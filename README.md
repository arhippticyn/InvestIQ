## 📊 About the Project

**INVEST_IQ** is a web application for personal finance management that allows users to track their income and expenses, analyze financial activity, and control their budget.

The application supports **Google authentication**, making the login process fast and secure. After signing in, users get access to a personal dashboard where they can add financial transactions, edit them, and view statistics of their income and expenses.

The project was developed **as part of a team**, with separate frontend and backend development.

---

## 🚀 Key Features

- 🔐 Sign up and login using **Google**
- 🛡 Secure authentication using **JWT**
- ➕ Add, edit, and delete **income and expense records**
- 📜 View **transaction history**
- 💰 Personal **budget management**
- 🗄 Secure **user data storage**

## 🏗 Architecture

The backend is implemented as a **REST API using FastAPI**, which processes client requests, handles user authentication, and interacts with the **PostgreSQL database**.

The frontend is built with **React and TypeScript** using a component-based architecture. **Redux Toolkit** is used for centralized state management across the application.
